# 🌸 BLUSHE - Beauty & Skincare Landing Page

A modern and elegant beauty product landing page built using **HTML**, **CSS**, and **Tailwind CSS**. The website showcases skincare products with a clean, feminine design focused on user experience and responsive layouts.

---

## 📸 Preview

BLUSHE is a cosmetic and skincare landing page featuring:

- Elegant Hero Section
- Product Showcase
- Brand Features
- Customer Reviews
- Pricing Plans
- Modern Footer
